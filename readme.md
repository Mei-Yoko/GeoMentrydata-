# GeoMentry Data Processing

This script processes geographical data using Pandas and GeoPandas. It reads a CSV file containing cell tower data, cleans the data, and maps it to geographical provinces.

## Requirements

- Python 3.x
- Pandas
- GeoPandas
- Shapely

You can install the required packages using pip: 
```
pip install Shapely
pip install GeoPandas
pip install Pandas
```
